#########################
Network Reference Designs
#########################

Netris can support any type of standard network design.

The majority of designs fall into one of four patterns:

#. Unmanaged Switch
#. Single L2 Domain
#. Collapsed Core
#. Data Center Leaf/Spine Designs (Modern High Throughput Redundant Design)
#. Data Center Core/Distribution/Access (Legacy High Throughput Redundant Design)
