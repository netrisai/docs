Netris |version| Release Notes (May/15/2025)
===============================================

What's new in Netris |version|?
-------------------------------

Features
--------
.. New capabilities visible to customers. One line each; optional 1–2 lines of context.

- **CloudStack support.**
- **Cumulus NVUE ACL support.**
- **Controller HA.**
- **Topology Validation.** Compare actual topology with designed topology in the controller and surface errors when mismatches are identified.
- **Global Routing.** Support native VPC subnet reachability from outside of the Netris-managed fabric.
- **Export Topology to CSV.** Topology information can be exported to NetQ ``.dot`` format and CSV format for labeling and patching cables.
- **Hypervisor Integrated Fabric.**
- **Optical Monitoring.** Light levels are now monitored.
- **EVPN-MH on Dell-SONiC.**
- **Add support for object labels.**
- **Add interfaces to V-Nets based on labels.**
- **Softgate PRO support.**
- **ACLs are now VPC aware.**
- **Netris controller can now be installed in an air-gapped environment.**
- **QoS for RoCE on EC-SONiC.**
- **EC-SONiC support.**
- **Ubuntu 24.04 support.**
- **Arista support.**

Enhancements
------------

- **CloudStack agent details.** Netris reports meta information about the Netris agent running on a CloudStack host.
- **CloudStack agent IPv6 SLAAC.** Adds support for IPv6 SLAAC.
- **CloudStack telemetry.** Adds agent telemetry view in **Network → Inventory**.
- **Spectrum-X improvements.** Support RoCE, Adaptive Routing, ASIC Monitoring, BGP graceful restart.
- **EC-SONiC support improvements.** Support QoS for RoCE, Adaptive Routing, L3VPN, VLAN-aware V-Net.
- **Dell-SONiC Topology Validation.**
- **Scale support for over 1000 VPCs and over 1000 DNAT rules.** Netris can now manage over 1000 V-Nets and over 1000 DNAT rules.
- **Underlay link type toggle and IP address change in Terraform on existing links.** When using Terraform, links can now be set to Underlay and back, and IP addresses can be changed without deleting and recreating links. Terraform will use the Update function.
- **Softgate HS supports Mellanox ConnectX NICs.** Softgate HS can now be installed on servers with Mellanox ConnectX network cards.
- **Optimize BGP overlay for Hypervisor Integrated Fabric.**
- **Improve Softgate PRO scalability.**
- **Load Balancer is VPC aware, supports IPv6, and is horizontally scalable.**
- **Improved breakout support.** 1x800G / 2x400G / 1x400G / 4x200G / 2x200G / 1x200G / 8x100G / 4x100G / 2x100G / 1x100G / 8x50G / 4x50G / 2x50G / 1x50G / 2x40G / 1x40G / 8x25G / 4x25G / 2x25G / 1x25G / 8x10G / 4x10G / 2x10G / 1x10G.
- **Improved interface stats aggregation.** Controller now shows bps, pps, and errors for aggregated interfaces based on data from member ports.
- **Collect VNI bps/pps stats for Softgate HS BGP peers.**
- **REST API changes.**
- **Dell-SONiC is now supported in CloudSim.**
- **Network links can be edited.**
- **Softgate HS Maintenance Mode.**

.. note::
   Some features are limited to specific platforms, as described in the :doc:`Netris supported functionality and platforms matrix <../supported-platform-matrix>`.
